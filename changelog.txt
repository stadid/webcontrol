Webcontrol by Stadid
Complete easy to use self-hosted standalone solution built on top of ebusd to replace Vaillant remote control system (MultiMATIC, SensoAPP, MyVaillant, etc.)

CHANGELOG

Notes:
(IMAGE) - update in the form of a full system install image
(CODE) - update via a ZIP file, installed via the program’s web interface or automatically via the Internet

+ functionality added
* functionality changed



Version 2023.08.08 (IMAGE)
=========================
First publicly available version.
Webcontrol development began in December 2022.
By the end of December 2022 an EBUS adapter was built and the first lines of code were written.
By March 2023 a working version with sufficient functionality was ready and beta-test by external user started.
In Summer 2023, along with improvements to the software functionality, an automated installation procedure was developed, and in August 2023 the first public version was released.

Version 2025.02.01 (IMAGE + CODE)
=========================
+ Added setting to display room temperature and humidity measured by the controller on the "Status" page (independently of zone assignment of the controller)
This is necessary for systems where controller is not assigned to any zone.

+ Added setting to make the "Parameters" page default. When enabled, accessing the program opens the "Parameters" page instead of "Status."

+ Added program update via Internet.

* Increased number of displayed heat-generator parameters on the "Parameters" page. In addition to the generator status code, temperatures of supply and return lines are shown (depends on heat generator model and firmware version).

+ Added support for heat generators in cascade — now the following parameters are displayed (status code, supply/return temperatures, and error code in case of a fault). If a generator faults, a notification is sent.

+ Telegram bot command "report"
The "report" command executes the "status" and "parameters" commands and returns a combined output from both.

* Improved time/date correction procedure for the controller — time/date are no longer corrected at 00:00 and 12:00 to avoid affecting cascade operation during switching.

[!] WARNING! This version is SETTINGS-COMPATIBLE (db file) with all previous versions EXCEPT 2023.08.08.
